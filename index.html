Grid:CSS Grid Layout, is a two-dimensional grid-based layout system with rows and columns, making it easier to design web pages without having to use floats and positioning. The CSS Flexbox offers a one-dimensional layout. It is helpful in allocating and aligning the space among items in a container

Bootstrap comes with a set of pre-styled responsive, mobile-first components that possess a definite UI kit.Sites created using Bootstrap follow the generic pattern that makes them look identical.

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages. It can be used as a toolkit for customizing the layout of different elements.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages
